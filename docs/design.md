# 设计规范

## 视觉方向
- Apple 气质启发，不做 1:1 复刻
- 主界面偏 `Finder / iCloud Drive / AirDrop` 的工具感
- 首屏节奏、留白和微动效借一点 `apple.com` 的高级感

## 主题 Token
- 浅色：冷白、雾灰、石墨灰
- 深色：蓝灰黑、石板灰
- 强调色：低饱和蓝，仅用于焦点、选中、链接和主要按钮
- 圆角：14 / 20 / 28 三档
- 阴影：柔和投影，不使用厚重浮夸的黑影
- 动效：160ms / 220ms，优先 `opacity + transform`

## 组件约束
- 按钮：主按钮使用低饱和蓝，次按钮使用轻描边与半透明底
- 输入框：弱边框、柔和背景、聚焦蓝色描边
- 分段控件：高密度 Apple 风格 segmented control
- 详情面板：参考 Quick Look 的大预览 + 紧凑属性编排
- 空状态：少字数、高留白、轻图形，不堆教程

## 响应式原则
- 桌面端采用三栏结构：操作区 / 共享流 / 详情
- 移动端改为单列堆叠，详情面板跟随内容区向下展开
- 不隐藏核心上传、预览、下载、删除能力，只调整布局密度
